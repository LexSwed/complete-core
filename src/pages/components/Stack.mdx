## Alignment

Since `Flex` can be used for rows and columns, use `alignAxisMain` and `alignAxisCross` to align items inside `Flex`
across main axis or cross axis.

`alignAxisMain="start"` (default)

<Playground>
  <>
    <Flex css={{ width: '400px', bc: '$lightBlue050' }} space="md" alignAxisMain="start">
      <ExampleBox css={{ width: '$16' }} />
      <ExampleBox css={{ width: '$12' }} />
      <ExampleBox css={{ width: '$20' }} />
    </Flex>
    <Flex css={{ width: '400px', bc: '$lightBlue050', mt: '$8' }} space="md" flow="row" alignAxisMain="start">
      <ExampleBox css={{ width: '$16' }} />
      <ExampleBox css={{ width: '$12' }} />
      <ExampleBox css={{ width: '$20' }} />
    </Flex>
  </>
</Playground>

`alignAxisMain="center"`

<Playground>
  <>
    <Flex css={{ width: '400px', bc: '$lightBlue050' }} space="md" alignAxisMain="start">

## Flow

You can reverse the components render using `flow="column-reverse"`

<Playground>
  <Flex flow="column-reverse" space="md">
    <ExampleBox>1</ExampleBox>
    <ExampleBox>2</ExampleBox>
    <ExampleBox>3</ExampleBox>
  </Flex>
</Playground>

And `row-reverse`

<Playground>
  <Flex flow="row" alignY="center" align="right" flow="row-reverse" space="md">
    <ExampleBox css={{ height: '24px' }}>1</ExampleBox>
    <ExampleBox css={{ height: '44px' }}>2</ExampleBox>
    <ExampleBox css={{ height: '64px' }}>3</ExampleBox>
  </Flex>
</Playground>

## Wrap

Control content wrapping via `wrap` property:

<Playground>
  <Box css={{ width: 50 * 4 + (4 - 1) * 8 }}>
    <Flex flow="row-reverse" wrap="wrap-reverse" space="sm">
      {Array(16)
        .fill(0)
        .map((el, i) => (
          <ExampleBox key={i}>{i + 1}</ExampleBox>
        ))}
    </Flex>
  </Box>
</Playground>
