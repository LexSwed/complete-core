# Picker

Picker is a component that mimics default `<select>` element.

<Playground>
  <Box css={{ p: '$8', bc: '$surfaceStill' }}>
    <Flex gap="md" css={{ width: '$40' }} cross="stretch">
      <Picker label="Item index" placeholder="1 number">
        {Array(100)
          .fill(null)
          .map((el, i) => (
            <Item key={i} value={`${i + 1}`} label={`Item ${i + 1}`} />
          ))}
      </Picker>
    </Flex>
  </Box>
</Playground>

## Controlled

<Playground>
  {() => {
    const [value, setValue] = React.useState('12');
    return (
      <Box css={{ p: '$8', bc: '$surfaceStill' }}>
        <Flex gap="md" css={{ width: '$40' }}>
          <Picker label="Item index" value={value} onChange={setValue}>
            {Array(20)
              .fill(null)
              .map((el, i) => (
                <Item key={i} value={`${i + 1}`} label={`Item ${i + 1}`} />
              ))}
          </Picker>
        </Flex>
      </Box>
    );
  }}
</Playground>

## Variants

Supports same `variant`s as `TextField` component.

<Playground>
  {() => {
    const [value, setValue] = React.useState('12');
    return (
      <Box css={{ p: '$8', bc: '$surfaceStill' }}>
        <Flex gap="md" css={{ width: '$40' }} cross="stretch">
          <Picker
            label="Item index"
            variant="underlined"
            placeholder="1 number"
            value={value}
            onChange={setValue}
            validity={Number(value) > 10 ? 'valid' : 'invalid'}
            hint={Number(value) > 10 ? '' : 'Oh no what did you do'}
          >
            {Array(20)
              .fill(null)
              .map((el, i) => (
                <Item key={i} value={`${i + 1}`} label={`Item ${i + 1}`} />
              ))}
          </Picker>
        </Flex>
      </Box>
    );
  }}
</Playground>
