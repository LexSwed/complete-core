## Validity

<Playground>
  {() => {
    const [value, setValue] = React.useState('me@email.com');
    const isValid = React.useMemo(() => /^\w+([\.-]?\w+)*@\w+([\.-]?\w+)*(\.\w{2,3})+$/.test(value), [value]);
    return (
      <Box css={{ p: '$6', bc: '$gray50' }}>
        <Flex space="md" flow="row">
          <TextField
            cross="stretch"
            validity={isValid ? 'valid' : 'invalid'}
            label="Email"
            hint={!isValid ? 'Check that your email is typed correctly' : ' '}
            placeholder="john.doe@mail.com"
            value={value}
            onChange={setValue}
          />
          <TextField
            cross="stretch"
            validity={isValid ? 'valid' : 'invalid'}
            label="Email"
            hint={!isValid ? 'Check that your email is typed correctly' : ' '}
            placeholder="john.doe@mail.com"
            value={value}
            onChange={setValue}
            variant="underlined"
          />
          <TextField
            cross="stretch"
            validity={isValid ? 'valid' : 'invalid'}
            label="Email"
            hint={!isValid ? 'Check that your email is typed correctly' : ' '}
            placeholder="john.doe@mail.com"
            value={value}
            onChange={setValue}
            variant="inline"
          />
        </Flex>
      </Box>
    );
  }}
</Playground>

## Label alignment

Since input is wrapped in `Flex` component, you can easily pass any props `Flex` supports,
for example to specify flow direction:

<Playground>
  <Box css={{ p: '$6', bc: '$gray50' }}>
    <Box css={{ width: '420px', tablet: { width: '300px' } }}>
      <TextField
        label="Enter your name"
        flow={{ initial: 'row', tablet: 'column' }}
        space={{ initial: '$6', tablet: '$1' }}
        placeholder="John Doe"
        hint="Real name will help your friends to find you!"
      />
    </Box>
  </Box>
</Playground>
