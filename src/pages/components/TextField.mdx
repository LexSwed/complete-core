## Validity

<Playground>
  {() => {
    const [value, setValue] = React.useState('me@email.com');
    const isValid = React.useMemo(() => /^\w+([\.-]?\w+)*@\w+([\.-]?\w+)*(\.\w{2,3})+$/.test(value), [value]);
    return (
      <Box css={{ p: '$6', bc: '$gray050' }}>
        <Flex space="md">
          <TextField
            validity={isValid ? 'valid' : 'invalid'}
            placeholder="Enter your email"
            value={value}
            onChange={setValue}
            label="Email"
          />
        </Flex>
      </Box>
    );
  }}
</Playground>

## Label alignment

Since input is wrapped in `Flex` component, you can easily pass any props `Flex` supports,
for example to specify flow direction:

<Playground>
  <Box css={{ p: '$6', bc: '$gray050' }}>
    <TextField
      label="Enter your name"
      flow={{ desktop: 'row', initial: 'column' }}
      space={{ desktop: '$6', initial: '$1' }}
      alignAxisCross={{ desktop: 'center', initial: 'start' }}
      placeholder="John Doe"
    />
  </Box>
</Playground>
