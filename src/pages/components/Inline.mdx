# Stack

`Stack` can also be used to align items in a row.

```tsx
import { Stack } from '@fxtrot/ui';
```

## Wrap

Control content wrapping via `wrap` property:

<Playground>
  <Box css={{ width: 50 * 4 + (4 - 1) * 8 }}>
    <Stack flow="row-reverse" wrap="wrap-reverse" space="sm">
      {Array(16)
        .fill(0)
        .map((el, i) => (
          <ExampleBox key={i}>{i + 1}</ExampleBox>
        ))}
    </Stack>
  </Box>
</Playground>

## Flow

You can reverse the components render using `flow="reverse"`

<Playground>
  <Stack flow="row" alignY="center" align="right" flow="row-reverse" space="md">
    <ExampleBox css={{ height: '24px' }}>1</ExampleBox>
    <ExampleBox css={{ height: '44px' }}>2</ExampleBox>
    <ExampleBox css={{ height: '64px' }}>3</ExampleBox>
  </Stack>
</Playground>
