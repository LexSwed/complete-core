## Wrap

Control content wrapping via `wrap` property:

<Playground>
  <Box css={{ width: 50 * 4 + (4 - 1) * 8 }}>
    <Inline wrap="wrap-reverse" flow="reverse" space="sm">
      {Array(16)
        .fill(0)
        .map((el, i) => (
          <ExampleBox key={i}>{i + 1}</ExampleBox>
        ))}
    </Inline>
  </Box>
</Playground>

## Flow

You can reverse the components render using `flow="reverse"`

<Playground>
  <Inline flow="reverse" space="md">
    <ExampleBox>1</ExampleBox>
    <ExampleBox>2</ExampleBox>
    <ExampleBox>3</ExampleBox>
  </Inline>
</Playground>
