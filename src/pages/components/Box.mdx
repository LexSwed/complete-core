## Box

`Box` is the main building block of the component library. It doesn't have any styles by itself,
but provides a way to pass any styles thanks to [`stitches`](https://stitches.dev/).

<Playground>
  <Box display="grid" gridTemplate="100px / 100px 100px" gap="$4" aria-selected="false">
    <Box bc="$gray200" textAlign="center">
      Styled
    </Box>
    <Box bc="$indigo600" textAlign="center" color="white">
      boxes
    </Box>
  </Box>
</Playground>
