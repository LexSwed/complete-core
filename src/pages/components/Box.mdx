## Box

`Box` component allows you to avoid creating a new component or styles object
(via [`css` prop](https://stitches.dev/docs/overriding-styles)) when you need just a few props,
for example a box with `padding`. It creates a `css` prop based on passed props,
so all the same rules as for `css` prop apply.

<Playground>

      Styled
    </Box>
    <Box bc="$indigo600" textAlign="center" color="white">
      boxes
    </Box>
  </Box>
</Playground>
